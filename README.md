# Qwen2 Translator App

Pequeña aplicación en Python que traduce frases de un idioma a otro utilizando el modelo **Qwen2-0.5B** de Hugging Face.

## Instalación
1. Clona este repositorio:
   ```bash
   git clone https://github.com/Carolina-Tendilla/qwen2-translator.git
   cd qwen2-translator
pip install -r requirements.txt
python translator.py
